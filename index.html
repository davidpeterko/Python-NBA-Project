NBA Shot Chart in Python

Who was it for?

This project was for myself and because I enjoy the NBA.

What was it?

A Python project that draw plots from the matlab library and overlays them using a join plot to create an NBA "Shot Chart" for the player Demarcus Cousins. Included with the .py file is pictures of the plots and the final shot chart created us DMC.jpg; include a bar graph on the sides of the court and color as well as a picture for player profile.

When was it?

Done in the summer of 2015; July.

Where was this?

This was done from home during my leisure.

Why was it done?

To learn Python (and the differences between 2 and 3) and how to use certain libraries. Also to help me understand how to manage packages with easy_install and pip for Python.

How was it done?

This DMCshot.py was created in VIM on my own laptop. I decided to use VIM because I enjoy learning new things and VIM really has increased my productivity by learning the shortcuts and having amazing plug ins.
